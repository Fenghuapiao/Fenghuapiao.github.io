Malicious code detection and analysis system [Code]

Today, malware has become a major threat to modern society, not only because of the increasing complexity of malware itself, but also because new malware is growing exponentially every day.In this project, we repeated the method of ensemble learning and used it to solve the problem of malware recognition.
